# ProjectBattle
2D battle simulator  

<b>Controls:</b>   
- Right-click = drag view  
- Left-click = set destination  
- Wheel = zoom  
- R = reset to right bottom corner of the map with initial zoom  
- O = toggle map overlay  
- P = toggle path lines  

<b>Implemented:</b>
- A-star path finding   
- Polygon rasterization    
- JavaFX graphics  

<b>Ongoing:</b>
- QuadTree optimisation  
- Unit collision
